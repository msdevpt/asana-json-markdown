# asana-json-markdown
A simple JSON to markdown convert for [Asana](https://github.com/Asana) export tasks file.

## Getting Started

```bash
$ git clone https://github.com/msdevpt/asana-json-markdown
$ cd asana-json-markdown\src
$ dotnet run in.json out.md
```
